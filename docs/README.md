## üìñ System Overview

The AHRP Report System consists of:
- **Backend**: Node.js API with Discord OAuth integration
- **Frontend**: React application with modern UI
- **FiveM Integration**: In-game reporting system
- **Database**: MySQL/MariaDB for data storage
- **Deployment**: Cloudflare Tunnel for secure public access
